# Falkon

__Falkon__ is a KDE web browser & It is built on the QtWebEngine.
![image](https://www.falkon.org/images/screenshot.png)

## Downloads

To download Falkon [click here](https://www.falkon.org/download/).

## Building

```sh
$ mkdir build && cd build
$ cmake ..
$ make && make install
```
